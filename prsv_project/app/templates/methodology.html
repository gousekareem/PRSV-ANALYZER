Methodology

Research Objective

This system is designed to detect Papaya Ring Spot Virus from papaya leaf images using a research-grade image processing pipeline supported by handcrafted feature engineering, SVM classification, severity estimation, heatmap-based explainability, and local RAG-grounded explanation generation.

1. Image Acquisition

Supports demo dataset mode, single upload, multiple upload, and ZIP batch upload with validation and Windows-safe storage.

2. Preprocessing

Performs resize, normalization, grayscale conversion, HSV conversion, denoising, and contrast enhancement.

3. Segmentation

Uses classical image processing for leaf isolation through HSV thresholding, contour detection, and morphological cleanup.

4. Symptom Emphasis

Generates edge maps, gradient response, Laplacian response, chlorosis emphasis, and symptom highlight overlays.

5. Feature Extraction

Extracts brightness, green ratio, hue mean, saturation mean, edge density, color variance, and entropy.

6. Classification

Uses an SVM classifier for Healthy vs Diseased prediction, with automatic fallback when trained artifacts are unavailable.

7. Severity Estimation

Computes infection percentage and severity label using weighted image-processing evidence and symptom coverage.

8. Explainability

Creates edge heatmaps and severity/symptom heatmaps for research presentation and visual evidence.

9. RAG Reasoning

Builds a retrieval query from image findings, retrieves PRSV knowledge chunks using TF-IDF similarity, and generates grounded explanation.

10. Reporting

Stores per-image JSON traces, visual artifacts, batch summary CSV/JSON, and downloadable run bundles for demos and papers.

Severity Logic

Severity is estimated using inverse green ratio, edge density, entropy, abnormal color score, and symptom region ratio. The result is converted into infection percentage and mapped to a research-friendly severity scale.

Future Extensions

This architecture is designed for future integration with CNN classifiers, multilingual farmer guidance, aphid detection, IoT-connected field analytics, mobile deployment, and full cloud-based agriculture decision systems.
